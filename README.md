# דשבורד — דמו

> ⚠️ **דמו** — כל הנתונים בפרויקט הזה (שמות פרויקטים, מספרים כספיים, עובדים) **בדויים לגמרי**.

**קישור חי**: https://dashboard-demo.dmaayan12.workers.dev — קוד כניסה: `123`

זהו עותק-דמו של דשבורד ניהול-סטודיו אמיתי שבניתי ומשתמש בו יום-יומי בעבודה. הדשבורד מתחבר במקור ל-monday.com ומציג תכנון פיננסי, מעקב אבני-דרך בפרויקטים לפי מדיניות תשלום, תחזית עומס-עבודה שבועית, ומעקב עומס-עובדים. בגרסה הזו, שכבת ה-backend מחזירה נתונים מומצאים במקום להתחבר למאנדי האמיתי - כך שאפשר להראות את המערכת בשלמותה בלי לחשוף שום מידע עסקי אמיתי.
הפיתוח שימש בתור MVP פונקציונאלי בלבד, ללא כניסה לUX/UI.

## מה כלול

- **באקלוג** - לוח בקרה פיננסי: תכנון מול ביצוע, מנגנון חוב/זיכוי, פרויקטים עם מדיניות-תשלום (אבני-דרך), פרויקטים במעקב בנק-שעות.
- **נפח עבודה צפוי** - תחזית עומס-עבודה שבועית לפי פרויקט, כולל זיהוי משימות חורגות.
- **עומס עובדים** - מעקב עומס חודשי בפועל לפי עובד.

בגרסא המקורית היו 4 שכבות הגנה:
1. תוסף כרום שנמצא כ"לא רשום" בגוגל
2. אפשרות להפעלת התוסף רק במידה ויש לך גישה למאנדי של החברה
3. בדיקת הרשאות למייל הספציפי בשרת עצמו
4. קוד כניסה לדאשבורד
