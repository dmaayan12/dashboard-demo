
**קישור**: https://dashboard-demo.dmaayan12.workers.dev — קוד כניסה: 123   (ממשק דסקטופ בלבד)

דשבורד אמיתי שבניתי לסמנכ"ל תפעול וכספים לצורך באקלוג פיננסי, ועבור ניהול הסטודיו והפרויקטים ברמת על. הדשבורד המקורי התחבר ל-monday.com.
> ⚠️ **דמו** — כל הנתונים בפרויקט הזה (שמות פרויקטים, מספרים כספיים, עובדים) **בדויים לגמרי**.
הפיתוח שמוצג כאן שימש בתור MVP פונקציונאלי בלבד, ללא שדרוג UX*UI שהגיע אחר כך.

## מה כלול

- **באקלוג** - לוח בקרה פיננסי: תכנון מול ביצוע, מנגנון חוב/זיכוי, פרויקטים עם מדיניות-תשלום (אבני-דרך), פרויקטים במעקב בנק-שעות.
- **נפח עבודה צפוי** - תחזית עומס-עבודה שבועית לפי פרויקט לניהול ספרינטים, כולל זיהוי משימות חורגות.
- **עומס עובדים** - מעקב עומס חודשי בפועל לפי עובד.

## בגרסא המקורית היו 4 שכבות הגנה:
1. תוסף כרום שנמצא כ"לא רשום" בגוגל
2. אפשרות להפעלת התוסף רק במידה ויש לך גישה למאנדי של החברה
3. בדיקת הרשאות למייל הספציפי בשרת עצמו
4. קוד כניסה לדאשבורד
